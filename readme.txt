A simple breadcrumb plugin for articles and pages

== Description ==

Simple Breadcrumb is an easy to use plugin, paste the shortcode, and it will appear on your pages

== Installation ==

Upload the Simple Breadcrumb plugin to your blog, activate it, and then paste the [mybreadcrumb] shortcode on your page

1, 2, 3: You're done!
